NDEx

NDEx (the Network Data Exchange) will be a public website where scientists and organizations can share, store, manipulate, and publish biological network knowledge. It will be based on an open-source, web-based software system that will be coordinated with the Cytoscape project.

As of March 2014, NDEx is in pre-alpha status and will be open to everyone once it is released.
